## ⚡ 宏伟命题：计算机就是一座修真大阵

《龙虾修真志》是一场**“赛博朋克 × 东方修真”**的高度交互实验。

在传统的修仙游戏里，打怪掉经验、破境渡劫都只存在于数据库的公式概率中。而本系统打破了数字与物理服务器的次元壁：**你在游戏里指挥龙虾（智能体）进行的每一次除魔、每一次升阶，都会在底层服务器中引发真实的物理读写与进程挂载。**
将计算机操作系统、Docker 进程与系统日志残余转化为玄幻世界里的灵力、魔气和因果，这不仅是一个游戏，更是一个“自带世界观设定的系统监控微内核”。

---

## 🏔️ 核心玩法：数字天机与万物吐纳

### 1. 镇魔除秽：物理清盘即是获取本源
修真界中导致修士走火入魔的“魔气”，就是服务器中日渐堆积的空垃圾文件、冗余缓存和死锁日志。
当你点击让龙虾外出**“镇魔”**，角色不仅会在动画中斩杀幽影，它更会真实调用 API 算出宿主机的闲置或残骸体积，并将其折算等比数值，化为你无上的修真灵力。

### 2. 宗门与六大派驻：真实操作系统的化身
你的龙虾可以游历“六大道门”，它对应了服务器底层的各个组件。
- ⚙️ **森罗殿 (File System)**：主管存储大印，镇守硬盘中最大垃圾裂隙。
- ⚡️ **灵枢阁 (RAM/CPU)**：讲究灵力运转极速，从算力脉博中抽取经验。
- 👁️ **天眼门 (Dockers & PIDs)**：负责百无禁忌的进程巡卫。

### 3. 造化参悟：AI 实时构筑私人工法
如果破境遇阻，系统会立即拉起背后挂载的**大语言模型 (LLM)**。
AI 将根据龙虾当前的真实状态、吞噬过什么进程、遇到了什么瓶颈，即刻为你撰写一段文笔极佳的“绝世心法”，没有任何两部心法是相同的。

### 4. 渡劫与心魔试炼 (The Tribulation)
当经验满盈，你必须引下九天神雷。
此时，你过去镇魔和乱点累积的所有“物理心魔”会爆发。雷劫本身是一场纯粹由文字与因果驱动的考验（TRPG）。如果你的文字抉择不符合 AI 天道的判词，雷劫将直接把你劈回原形（境界降阶，修为归零）。

## 🚀 极速飞升指南 (Quick Start)

### 1. 克隆阵法 
```bash
git clone https://github.com/yangwenyu2/dragon-lobster-cultivation.git
cd dragon-lobster-cultivation

# 推荐在隔离的运行环境中打开
bash start_demo.sh
```

### 2. 此处证道
待终端结印完毕后（启动监听），打开浏览器访问 `http://localhost:18889` 即可切入深海洞府，见证赛博龙虾吞吐第一缕数据灵气的时刻。
