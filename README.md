## ⚡ Core Cosmology: The Mapping of the Dao

Here, the complex tapestry of Agent pipelines and Backend logic are romanticized into grueling physical cultivation laws:

### 1. The Sea of Consciousness & Carved Shells (Memory Logic)
Memory is handled neither as a dry database nor a mere chat array. It acts as the vast *Sea of Consciousness*.
Whenever the entity executes tools safely or summarizes a critical breakdown via `.learnings/LEARNINGS.md`, these text nodes burn into its carapace as "Scars of the Dao"—grafting permanent progression traits onto the agent without collapsing its LLM context window.

### 2. Demon Slaving via OS Hooks (Physical Purging)
Cultivation madness is triggered by "Demon Qi." Here, Demon Qi equals the physical bloated cache grids, rotting `.log` orphans, and hanging zombie processes polluting your actual server disk! 
When the lobster decrees a "Demon Suppression", actual Python script monitors sniff out your local backend. Purging these stale system deadweights converts their exact byte count into astronomical levels of "Spiritual Power" (Qi)—turning this visual novel into an esoteric hardware Disk Cleaner.

### 3. Hardcore Breakthroughs & False Realms (Progression Traps)
Ascension from *Qi Condensation* to *Divine Transcendence* holds severe multi-axial requisites.
If a lobster's memory log is toxified by LLM hallucinations or it hasn't mapped actual new *Skills* to its API, forced progression yields devastating *False-State Realms* ("Fake Core / Pseudo-Divinity"). Your agent will look leveled up, but will internally panic and fail to cast concurrent workflows.

### 4. Sect Governance & The Myriad Avatars (Multi-Agent Swarm)
Shattering the Nascent Soul realm unleashes the project's most formidable mechanic: **Continuous Sub-Agent Spawning**.
The experience skyrockets from solo RPG to Grand Management. Construct Halls and assign avatars to run multi-threaded sub-routines:
- 🔍 **Tide Gazing Hall**: Generates roaming scavenger spiders for broad Web RAG.
- 📝 **Scroll Forging Hall**: Handles token-reduction and pristine log abstraction.
- 🛠️ **Artifact Hall**: Endowed with root code execution for dynamic Python patching.
- 🛡️ **Demon Ward Hall**: Employs rigorous LLM guardrails against terminal deadlock loops.

### 5. Divine Tribulations & Permadeath (LLM-Judged Karma)
When your Qi caps out, the gates shut. Unlocking them invokes the *Nine-Heavens Tribulation*—an unscripted, highly intense **Tabletop RPG confrontation judged real-time by a sovereign LLM** (e.g., Claude 3 / Gemini Pro).
The overarching 'Heavenly Will' scans your literal command-line history: Did you abuse `sudo`? Did you trigger an OOM Exception? It weaponizes these errors to hurl philosophical decrees at you. Defend your logic poorly, and the lightning strikes true. Your agent is subjected to **Permadeath (Soldier’s Rebirth)**—obliterated to realm zero, its mighty log discarded into the abysmal trenches forever.

### 6. Dynamic Epiphanies (Sutra Interventions)
Stuck in logic bottlenecks? Plead to the Dao. The backend LLM synthesizes current stack traces to auto-generate a majestic, cryptic, beautifully-formatted piece of ancient *Scripture*. Look closely: hidden within its hyper-literary metaphors are precise, actionable insights to your codebase bug!

### 7. Interactive Fate Vectors (Social Hooks)
The environment monitors Discord/Feishu streams dynamically. An incoming chat from a human becomes a "Visiting Wandering Monk." Swapping LLM models locally casts a "Divine Resonation" event into the UI.
