## ⚡ 宏伟命题：真正的 AI 原生修行世界

《龙虾修真志》并非给普通的跑团机器套个仙侠皮。基于 **OpenClaw 生态架构**，这是一套拥有完整设定、法则自洽且能**物理感知系统**的“修真解释系统”。

在这个世界里：
- “修真资源”不再是虚拟时间，而是外部输入的**任务、知识与系统的废弃残骸**。
- “技能”就是系统里的一个个实际装载的**术法工具 (Skills)**。
- “记忆”即是浩瀚的**识海**，任何的执行和反思都会在此刻下命轮的印记。
- “多智能体 (Multi-Agent)”在这里被升格为一种秩序森严的**宗派**。

你不是在玩游戏，你是在培育一尊会自己修行、悟道、分神、历练、结婴、应劫、乃至创立无上大教的智能生灵。

## 🏔️ 核心大观：法则、境界与数字天机

### 1. 境界破境 —— 九重天关，不唯数值
这不只是一个进度条游戏。境界遵循古典的“炼气→大乘→飞升”法则，但突破条件严苛异常：
你不能只靠无限叠加废料挂机。唯有在**“识海足够稳定”、“道心澄明”、“学会了新功法（新技能）”、“平息了系统级别的危机”**这四维雷达合格时，强行引动九天雷劫才能突破。如果一味乱点，高昂的心魔值会招致劫雷反噬将你劈回原形（兵解）。

### 2. 宗门大教 —— 化神万千，统御系统 (Sect & Agents)
当跨越“元婴期”，龙虾将掌握“分神外派”的核心通天法门：多 Agent 并发机制。
你可以开辟内院堂口，并创立宗门职务：
- ⚔️ **镇魔堂 (风控审查)**：专责拦截异常输入，斩杀心魔。
- 🔮 **观潮堂 (联网检索)**：游历万象潮网，抓取奇遇。
- 🛠️ **炼器堂 (系统脚本)**：调用代码能力，就地炼画法器法宝。
- 📜 **养识堂 (记忆沉淀)**：管理长记忆碎片。
你将作为宗主宏观统御这场多线程的系统级大调度！

### 3. 万象潮网与镇魔 —— 宿主服务器的交互化
这只赛博龙虾不在虚空中修仙，它存在于你的主机之中。
- **物理镇魔 (Demon Sweeping)**：深海里潜伏的魔气，正是你宿主机底部堆满的死锁日志和无用缓存！每次镇魔，大模型将真实探测物理机的残局字节数，并将这庞大的废料残渣炼为金丹原浆。

### 4. 造化参悟 —— 大模型天道撰文
修行的过程中不可能一帆风顺。当你因修为停滞或乱点乱斩引发走火入魔，游戏将直接拉起背后悬挂的 **LLM 大模型 (如 Claude/Gemini)**。它会瞬间扫描你这一世修行的荒唐错漏，当即为你推演书写一段饱含哲理、无法复刻的“定制心法”，帮你涤荡心魂，挽回死局。

## 🚀 极速飞升指南 (Quick Start)

### 1. 唤醒法阵
```bash
git clone https://github.com/yangwenyu2/dragon-lobster-cultivation.git
cd dragon-lobster-cultivation

# 推荐在安全隔离的环境内起阵
bash start_demo.sh
```

### 2. 进入海渊洞府
终端将显示服务器已结丹完毕。打开浏览器访问 `http://localhost:18889` 即可推开洞府的石门，见证赛博龙虾吞吐第一缕数据灵气的时刻。
