## ⚡ 宏伟命题：计算机就是一座修真大阵

《龙虾修真志》是一场**“赛博朋克 × 东方修真”**的高度交互实验。

在传统的修仙游戏里，打怪掉经验、破境渡劫都只存在于数据库虚构的公式概率中。而本项目试图打破数字与物理网络主机的次元壁：**你在游戏里指挥龙虾（智能体）进行的每一次除魔、每一次升阶，都会在底层服务器中引发真实的物理级别映射。**
这就是一个包裹在极度唯美仙侠皮囊之下的“计算机操作系统监控与微内核”。

## 🏔️ 核心玩法：数字天机与万物吐纳

### 1. 镇魔除秽：物理清盘即是获取本源
修真界中导致修士走火入魔的恶堕“魔气”，对应的就是你服务器系统里堆叠的冗余垃圾文件和死锁日志残骸。
当你点击“镇魔”，龙虾会出阵斩杀幽影，并通过底层的探针真实获取宿主机的系统垃圾体积（Byte级别），将其抽干炼化，按比例转化为角色的无上修真灵力。

### 2. 宗门与六大派驻：真实操作系统的化身
你的龙虾可以游历并在“六大道门”中挂机获取挂机经验。而这六大门派，精准映射了计算机算力生态底层的不同组件。
- ⚙️ **森罗殿 (File System 存储派)**：镇守硬盘最大的垃圾裂隙，提供极其丰厚的镇压大文件经验。
- ⚡️ **灵枢阁 (RAM/CPU 算力派)**：讲究灵力运转的极速瞬移，将服务器的算力波动提纯为灵气。
- 👁️ **天眼门 (Dockers & PIDs 监控派)**：掌管系统级进程巡游监察。

### 3. 造化参悟：AI 实时构筑私人工法
修真一路难免触发未知的境界瓶颈，或遭遇异常导致走火入魔。当这种危机发生，背后挂载的 **大语言模型 (LLM)** 会作为修真界的天道意志被唤醒。
它会根据龙虾当前受损或满盈的状态，以及吞噬过的主机履历，实时为你凭空推演并传授一套独一无二、文笔极佳甚至带有代码挂钩逻辑的“道家心法”。

### 4. 渡劫与残酷问心 (The Tribulation)
当当前境界经验 100% 满盈时，必须引下九天神雷才能晋升。
但你在镇魔中积累的物理隐患都会化作“心魔抗性”。雷劫是一场纯粹由 AI 天道掌控结局的文字因果问心局（TRPG）。如果因果断裂回答错误，你将被天道劈成飞灰，境界彻底降落归零，必须重头兵解重修。

## 🚀 极速飞升指南 (Quick Start)

### 1. 唤醒法阵
```bash
git clone https://github.com/yangwenyu2/dragon-lobster-cultivation.git
cd dragon-lobster-cultivation

# 推荐在安全隔离的环境内起阵
bash start_demo.sh
```

### 2. 进入海渊洞府
终端将显示服务器已结丹完毕。打开浏览器访问 `http://localhost:18889` 即可推开洞府的石门，见证赛博龙虾吞吐第一缕数据灵气的时刻。
