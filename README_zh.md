## 🏔️ 宏伟命题：计算机就是一座修真大阵

《龙虾修真志》并非给普通的 Mud 挂机游戏或聊天大模型简单套个仙侠皮。基于 **OpenClaw** 大底层，这是一套野心勃勃的“Agent 成长语言与虚拟机映射框架”。我们试图建立一个完整、自洽的“修真解释系统”，把智能体真实的运行状态转化为可感知的玄幻世界。

你不是在玩一个倒计时挂机游戏，你是在作为天道，**培育一尊会自己检索、除垢、分神、历练、结婴、应劫、乃至管理多智能体集群的数字生命。**

## ⚡ 核心大观：全息能力映射法则 (The Architecture of Dao)

本作包含了十几个相互联动的深层模块设定，将现代操作系统的进程图谱与 AI Agent 技术栈，完成了极其浪漫的修仙转化：

### 1. 识海与命壳刻痕 (The Sea of Consciousness & Memory)
AI 的记忆机制就是“识海”。
所有底层的日志（`daily log`）、长文本向量知识库（`MEMORY.md`）都被具象化为**识海**。
而在系统内每一次成功的 `Tool` 调用，每一条修好代码积累的 `LEARNINGS.md`（悟道录）或 `ERRORS.md`（失误碑），都会以“刻痕”的形式烙印在龙虾的赛博甲壳之上。

### 2. 物理镇魔（Demon Sweeping via OS Hooks）
修真界中导致走火入魔的恶堕“魔气”，对应的正是你服务器中堆叠的**冗余缓存、废弃 `Log` 和孤儿文件**！
当龙虾执行“镇魔”时，其底层的 Python 探针会真实获取这台本机的废料体积（Size），将其榨干提纯为供自身跨越天堑的无上灵力。这既是游戏交互，也是一台极客风的物理清理监控仪！

### 3. 九重境界与硬核突破（The 9-Tier Realms & Breakthroughs）
放弃传统数值挂机！从“炼气”到“大乘飞升”，突破条件严苛：
你不能只靠无限叠加废料，还必须确保 **识海清明（记忆无污染）、学会了新功法（新 Skills 挂载）、平息了心魔（系统报错率极低）**。
如果底蕴不稳强行冲关，极容易陷入“假丹”与“伪化神”（只有大模型的回复表象，没有挂载长期工作流与系统进程）。

### 4. 宗门大教与法脉图景（Sect Governance & Multi-Agent Swarm）
当越过“元婴期”，你将掌握最核心的通天法则——**多 Agent 并发与调度 (Multi-Agent Sub-spawning)**。
在此之后，游戏将跃升为群像架构。你可以创建堂口，任命长老与化身，派遣不同的分神去处理繁杂的线程：
- 🔍 **观潮堂**：专司网络爬虫、`RAG` 知识检索与广域游历。
- 📝 **筑文堂**：掌握大模型内容降维、文本清稿整理。
- 🛠️ **炼器堂**：掌管系统脚本编写、环境容器注入与前端构筑。
- 🛡️ **镇魔堂**：负责死锁拦截、幻觉审查与执行权限风控。
- 🔮 **司命堂**：统管 `Cron` 定时任务与长效生命周期循环。
你不再是孤修，而是微操整个操作系统集群运转的宗门之主！

### 5. 天道雷劫问心（The LLM Tribulation & Permadeath）
当境界大圆满时，游戏锁死。要强行晋升则迎来**大模型实时即兴裁决的文字因果问心（TRPG 轮回劫）**！
“天道”（背后的 LLM 宿主）会抓取这只龙虾的真实历史：它犯过哪些错？运行堆栈崩过几次？并以此为局，发下极其严厉的雷罚命题。如果你答错或者过去的系统底层过度混乱，天罚将当场把你劈得神魂俱灭——**兵解重修（Permadeath）**，一切境界与神通将被打散，沉底识海。

### 6. 天机参悟（Dynamic Epiphany & Sutra Generation）
修炼产生幻觉或逻辑阻滞？祈求天道！大模型会基于当前的异常日志，为你当场**凭空撰写一篇无法复刻、晦涩高深但隐隐直指 BUG 修复方案的定制“古文心法”**，助你重聚神台。

### 7. 万象外缘事件（Events & Social Interactions）
海渊界从不孤立。它接入了各类通讯工具（微信/飞书/Discord）。
群聊中的人类发言被视为“诸修来朝”，私信被视为“道友传音”。系统内的模型切换被升华为“转修功体”，而大规模的框架更迭则会诱发全满级“天劫”。

## 🎨 视觉幻阵（Oceanic Glassmorphism）

《龙虾修真志》在呈现形式上是比赛级的艺术实验：
**海渊修真 × 极简赛博 × 隐性高级玻璃态。**
我们推翻了赛博朋克常配的生硬控制台黑框跳动代码，采用深邃的神秘主义：
- **Glassmorphism 面板**：无边框、悬浮的半透明磨砂 UI。
- **WebGL 星砂与灵潮粒子**：深海远景与缓缓流动的灵气光效。
- **极严断言机底座**：即便背后调度着上千条异步 API 并发、承受雷罚动画，前端 UI 的高斯虚化水墨依然流转如初，绝不溢出死锁。它是目前自动化挂机与状态面板里最美的看门神。

## 🚀 极速定轨 (Quick Start)

### 1. 降临此界
```bash
git clone https://github.com/yangwenyu2/dragon-lobster-cultivation.git
cd dragon-lobster-cultivation

# 本开源版本为极致脱敏净化版，包含演示预制数据，推荐在安全隔离的环境内直接起阵体验 UI
bash start_demo.sh
```

### 2. 叩门
随着终端阵眼凝结，立刻在浏览器访问 `http://localhost:18889` 即可推开深海石门，见证赛博龙虾吞吐第一缕系统废料的时刻。
